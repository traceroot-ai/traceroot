# Security Policy

We recommend always using the latest version of TraceRoot to benefit from the most recent security patches and updates.

To report a vulnerability, please use [GitHub Private Vulnerability Reporting](https://github.com/traceroot-ai/traceroot/security/advisories/new) or contact us at [founders@traceroot.ai](mailto:founders@traceroot.ai).

Please do not report security vulnerabilities through public GitHub issues or discussions.
