---
title: Supported Models
description: All LLM models available natively in Traceroot Cloud
---

Traceroot Cloud provides access to the following models out of the box — no API key required. These are the system models available when using the AI agent without [BYOK](/ai-agent/byok).

## Anthropic

| Model | Description |
|-------|-------------|
| `claude-opus-4-6` | Most capable model for complex reasoning and debugging |
| `claude-sonnet-4-6` | Balanced performance and speed |
| `claude-sonnet-4-5` | Previous Sonnet generation |
| `claude-haiku-4-5` | Fast and lightweight |

## OpenAI

| Model | Description |
|-------|-------------|
| `gpt-5` | OpenAI's most capable model |
| `gpt-5-mini` | Fast and cost-efficient GPT-5 variant |
| `o3` | Advanced reasoning model |
| `o4-mini` | Lightweight reasoning model |

---

<Note>
Want to use a model not listed here? Set up [BYOK](/ai-agent/byok) to bring your own API key and unlock any provider — including Azure OpenAI, Google Gemini, Amazon Bedrock, DeepSeek, and OpenRouter.
</Note>
