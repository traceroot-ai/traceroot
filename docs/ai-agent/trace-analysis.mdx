Ask the AI agent questions about your traces directly from the chat interface in the TraceRoot dashboard. The agent can search, filter, and aggregate across your full trace history.

## Example Queries

**Errors**
- "Summarize the errors in the last hour"
- "What's the most common failure mode for this agent?"
- "Show me all traces where the search tool failed"

**Performance**
- "Which agent has the highest latency?"
- "What's the average cost per trace this week?"
- "Compare latency between gpt-4o and gpt-4o-mini calls"

**Data exploration**
- "How many traces did user-123 generate today?"
- "What's the total token spend this month?"
- "Show me a breakdown of model usage across all traces"

## Tips

- Be specific about time ranges — "last hour" is better than "recent"
- Mention specific services or agents to narrow the search
- Ask follow-up questions — the agent maintains context within a conversation
