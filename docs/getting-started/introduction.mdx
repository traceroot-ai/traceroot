Traceroot is an open-source observability and self-healing layer for AI agents. Instrument your code with a few lines, get full visibility into every LLM call, tool invocation, and agent decision — then let Traceroot's AI agent debug failures and create fix PRs automatically.

## Key Capabilities

  <Card title="Tracing" icon="route">
    Capture every LLM call, tool use, and agent step with OpenTelemetry-based instrumentation.
  </Card>
  <Card title="AI Debugging" icon="bug">
    An AI agent that performs root cause analysis in a sandboxed environment using your actual source code.
  </Card>
  <Card title="Self-Healing" icon="wrench">
    Automated fix PRs created directly on GitHub, linked back to the trace that surfaced the issue.
  </Card>

## How It Works

1. **Instrument** — Add the Traceroot Python SDK to your application. Auto-instrumentation captures LLM calls from OpenAI, Anthropic, and LangChain.
2. **Observe** — View traces in the dashboard. See the full span hierarchy, token usage, cost, and errors.
3. **Debug** — Ask the AI agent to analyze failing traces. It clones your repo into a sandbox, reads the relevant source code, and identifies the root cause.
4. **Fix** — The agent writes a fix and opens a PR on GitHub with a description linking back to the original trace.
