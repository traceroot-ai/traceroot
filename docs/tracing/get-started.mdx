## 1. Create a Project

Sign up at [app.traceroot.ai](https://app.traceroot.ai) and create a project. Copy your API key from **Settings → Access Keys**.

## 2. Install the SDK

<CodeGroup>
```bash pip
pip install traceroot
```

```bash uv
uv add traceroot
```

```bash poetry
poetry add traceroot
```
</CodeGroup>

## 3. Set Environment Variables

```bash .env
TRACEROOT_API_KEY=your-api-key
TRACEROOT_SERVICE_NAME=my-agent
```

## 4. Initialize Tracing

Call `traceroot.initialize()` at the top of your entry point, before importing LLM providers:

```python
import traceroot
from traceroot import Integration

traceroot.initialize(
    integrations=[
        Integration.OPENAI,      # auto-instrument OpenAI
        Integration.ANTHROPIC,   # auto-instrument Anthropic
        Integration.LANGCHAIN,   # auto-instrument LangChain
    ],
)
```

<Tip>
Only pass integrations for the libraries you actually use. Unused integrations are skipped automatically.
</Tip>

## 5. Instrument Your Code

Use `@observe` to wrap your agent functions:

```python
from traceroot import observe
import openai

client = openai.OpenAI()

@observe(name="my_agent", type="agent")
def run_agent(query: str) -> str:
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": query}],
    )
    return response.choices[0].message.content

result = run_agent("Summarize the latest news")
```

The `@observe` decorator creates a parent span. The OpenAI call inside is captured automatically as a child span.

## 6. View Your Traces

Open [app.traceroot.ai](https://app.traceroot.ai), navigate to your project, and you'll see the trace with the full span hierarchy, token usage, cost, and timing.
