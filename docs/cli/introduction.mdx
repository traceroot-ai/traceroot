The TraceRoot CLI (`traceroot`) is a small, read-only client for the TraceRoot public API. Authenticate once with your project API key, then list, inspect, and export traces directly from your terminal — no web app required.

It's built for the debugging loop: copy-pasteable trace IDs, readable span trees, and JSON output that pipes cleanly into `jq` and scripts. The CLI never instruments your code or emits spans — that's what the [SDKs](/tracing/get-started) are for.
