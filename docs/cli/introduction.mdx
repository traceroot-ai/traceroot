The TraceRoot CLI (`traceroot`) is a small, read-only client for the TraceRoot public API. Authenticate once with your project API key, then list, inspect, and export traces directly from your terminal — no web app required.

You get readable span trees, durations, and errors inline — or JSON output for piping into scripts. The CLI only reads traces; instrumenting your app to produce them is the [SDKs'](/tracing/get-started) job.
